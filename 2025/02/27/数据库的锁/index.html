数据库的锁

锁

全局锁

让数据库变成read only

表级锁

  1. 表锁：给表加锁，同时本线程无法访问其他表
  2. 元数据锁：对表操作隐式使用
  3. 意向锁：行级锁之前，快速判断有无记录被加锁
  4. AUTO-INC锁：用于自增类型的加锁，不然容易出错

行级锁

  1. Record Lock：锁记录
  2. Gap Lock：锁范围（开区间）。间隙锁之间是兼容的的，可以由多个事务持有重复区间的间隙锁。
  3. Next-Key Lock：记录锁+间隙锁（一开一闭）。既保护了当前记录，又阻止间隙之间有新值插入。
  4. 插入意向锁：用于插入时占用锁等待队列。

分布式锁
